# huntflow-api-client

### About
Example python client for [huntflow-api](https://github.com/huntflow/api)

Made for test assignment

### Requirements
* xlrd
* click
* pandas
* tinydb 
* colorama
* requests 

`pip install -r requirements.txt`
### Usage
`huntflow-api-client.py --token <your token> --folder <folder with files>`
